# pickleball-tactical-analysis

JINHOPA 匹克球戰術分析工具，已升級為 React + TypeScript + Vite。

## 開發

```bash
npm install
npm run dev
```

## 建置

```bash
npm run build
```

主要程式碼在 `src/App.tsx`，樣式在 `src/styles.css`。首頁 `index.html` 只顯示導覽列，戰術分析工具在 `tactical-analysis.html`。既有靜態頁面與 SEO 檔案放在 `public/`，建置時會一併輸出。
